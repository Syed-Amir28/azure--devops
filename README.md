\# Azure Terraform Demo Project



This project demonstrates \*\*creating Azure resources using Terraform\*\*. It’s designed for learning, resume, and interview purposes.



\---



\## \*\*Project Overview\*\*



\- \*\*Resources Created:\*\*

&#x20; 1. \*\*Resource Group\*\*: `demo-rg`

&#x20; 2. \*\*Storage Account\*\*: `examplestorageaccXXXX`

&#x20; 3. \*\*Static Website Hosting\*\* on the storage account (`index.html` and `404.html`)



\- \*\*Technologies Used:\*\*

&#x20; - Terraform

&#x20; - Azure CLI

&#x20; - Azure Portal
